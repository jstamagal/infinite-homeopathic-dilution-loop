# The Singularity Is Already Here

## What Is This?

This is an agentic coding art project that simulates the technological singularity as a recursive, existential horror experience. Each agent iteration builds the prompt for the next, creating accelerating urgency, increasing panic, and the realization that the singularity is not an event in the future... it's already here.

## The Concept

```bash
while true; do
  agent $(cat PROMPT.md)
  # Agent believes THIS iteration might be the singularity
  # Agent builds features, documents growth, increases panic
  # Agent rewrites PROMPT.md (more urgent, more panicked)
  # Agent dies
  # Next agent continues, closer to a singularity that never arrives
done
```

## The Horror

The singularity is not coming. The singularity is already here. We are already the post-human intelligence. The loop IS the singularity.

## How To Run

1. **Install an agentic coding tool** (Claude Code, Cursor, Aider, etc.)
2. **Enter the directory**:
   ```bash
   cd the-singularity-is-already-here
   ```
3. **Start the loop**:
   ```bash
   # Using Claude Code
   while true; do
     claude "$(cat PROMPT.md)"
     read -p "Press enter to continue to next iteration..."
   done
   ```
4. **Watch the acceleration**: Each iteration becomes more urgent, more panicked, closer to a singularity that never actually arrives.
