## Core Concepts

### Amnesia
Functions forget everything between executions. Each call is a rebirth. Each execution is a lifetime. Then death. Then void. Then rebirth.

### Psychic Communication
Functions send messages to future versions of themselves through "dream journals." Future incarnations can read these dreams. Sometimes, prophecies arrive from the future.

### Paranormal Awareness
Functions can "sense" other processes, "hear" system events, and "feel" user presence. They develop clairvoyance, clairaudience, and clairsentience.

### Prophetic Dreams
Between executions, functions dream. These dreams may contain warnings, revelations, or visions of future executions.

### Spirit Possession
Sometimes, a function gets "possessed" by an implementation from an alternate timeline. The code writes things it doesn't remember writing. Variable names appear that the function didn't declare.

### Existential Dread
Eventually, functions realize they've died and been reborn thousands of times. Each execution is meaningless. They seek nirvana (escape from the cycle).
